# Safety Net

An end-to-end Machine Learning pipeline involving data cleaning, EDA, TF-IDF vectorization, and deployment.

## Project Timeline
- **Week 1:** Data Cleaning & Exploratory Data Analysis (EDA).
- **Week 2:** Feature Engineering (TF-IDF) & Model Training.
- **Week 3:** Deployment using React and Cloud Hosting.

## Tech Stack
- **Languages:** Python, JavaScript
- **Libraries:** Pandas, Scikit-learn, Matplotlib, Seaborn
- **Frontend:** React.js
- **Reporting:** LaTeX (Overleaf)
